Quick help
  • Supported operators: AND, OR, NOT (case-insensitive)
  • Parentheses for grouping: (title:sceatta OR description:sceatta) AND collection:"Wiltshire Museum"
  • Fielded terms: collection:"Wiltshire Museum" or collection:Wiltshire
  • Phrase search: use double quotes
  • Field map lets you translate friendly names to API names (one per line, colon-separated)
